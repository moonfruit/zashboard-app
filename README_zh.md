# Zashboard

[Zashboard](https://github.com/Zephyruso/zashboard) 的桌面应用包装，使用 [Wails](https://wails.io/) 构建。

[English](README.md)

## 关于

Zashboard 是一个基于 Clash API 的 Dashboard 面板。本项目将其打包为轻量级的原生桌面应用，支持 Windows、macOS 和 Linux 平台。

## 功能特性

- 原生桌面体验
- 跨平台支持 (Windows / macOS / Linux)
- 轻量级，资源占用低
- 与 Web 版本功能一致

## 构建

### 前置要求

- [Go](https://go.dev/) (stable)
- [Node.js](https://nodejs.org/) (LTS)
- [pnpm](https://pnpm.io/)
- [Wails CLI](https://wails.io/docs/gettingstarted/installation)

### 开发模式

```bash
wails dev
```

### 构建生产版本

```bash
wails build
```
